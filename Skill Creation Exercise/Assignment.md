# SKILL CREATION EXERCISE

Create a Photography Skill (for use with Nano Banana Pro) with the Skill Creator Skill in Claude Chat

1. Open Claude App
2. Choose 'Customize'
3. Select 'Create new skills'
4. Press '+' in the 'Skills' column
5. Choose 'Create with Claude'
6. Add the file 'Photography Skill Input Gemini.md' to the chat
7. Follow instructions from there
8. When asked, choose 'Test and refine' for better results

Notes: 

- Beware of famous SA photographer references!
- Make sure the entire workflow is tested!
- IP may be blocked!
